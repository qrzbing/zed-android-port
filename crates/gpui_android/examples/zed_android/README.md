# Zed on Android — example application

The full project README is at the **[repo root](../../../../README.md)**.

This directory is the example application that composes Zed's `Workspace` /
`Project` / `MultiWorkspace` / `Editor` / `Search` / `GitPanel` /
`GitGraph` / `Extensions` / `Terminal` crates and bundles them as an
Android APK via Gradle.
