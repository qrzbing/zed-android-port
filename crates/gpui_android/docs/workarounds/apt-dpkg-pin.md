# Apt dpkg pin

**Status:** Active

$PREFIX/etc/apt/preferences.d/zed-pin-dpkg sets dpkg + dpkg-deb at Pin-Priority 1001 against any upstream version. Stops apt install <pkg> from clobbering our patched dpkg with the upstream binary that has com.termux paths baked in.
